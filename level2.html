▶ THEORY

What is Brute Force?

A brute force attack tries many passwords systematically until the correct one is found. It exploits weak or common passwords.

Types of Brute Force

Pure Brute Force — tries every combination (slow)
Dictionary Attack — uses a wordlist of common passwords
Credential Stuffing — uses leaked username:password pairs

Common Tools

hydra -l admin -P rockyou.txt http-post-form
burpsuite intruder → sniper mode
wfuzz, ffuf, medusa

How to prevent

Rate limiting, CAPTCHA, account lockout after N failed attempts, MFA, and strong password policies all prevent brute force.

Wordlist location (Kali)

/usr/share/wordlists/rockyou.txt
